<h3>LAL is a Discord bot made using the AKAIRO framework!</h3>

## First time setup

Clone the project

```bash
  git clone BluePotatoBP/LAL-Akairo
```

Go to the project directory

```bash
  cd LAL-Akairo
```

Install dependencies

```bash
  npm install
```

Run the application

```bash
  npm run start / dev (nodemon)
```
